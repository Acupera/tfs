## Getting Started

### Installation

    npm install tfs
    npm link

`npm link` allow you to use `tfs` command anywhere within you file system.

### Commands

* __changeset__ [file(s)] &lt;options&gt;
* __checkin__ [file(s)] &lt;options&gt;
* __get__ [file(s)] &lt;options&gt;
* __undo__ [file(s)] &lt;options&gt;
